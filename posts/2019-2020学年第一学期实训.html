    实训

    # 2019年12月23日 第十七周 周一 实训第一天

    1. 安装Node.js。

    2. 安装Typora。

    3. 安装Jquery。

    4. 注册Github、代码托管器。

    5. 学习菜鸟教程、http://www.runoob.com/git/git-tutorial.html。

      注:

      • Ctrl+/是查看源代码。

      • 保存文件后缀名为md。

      • 代码符号输入后必须加空格。

    # 2019年12月24日 第十七周 周二 实训第二天

    1. 安装Git(上传工具)。

    2. 打开 www.github.com 创建新的代码库，命名要求:用户名+github.io。

    3. 创建工作文件夹，在文件夹中右键打开Git Bash Hear、输入git clone+粘贴代码库地址。

    4. 在文件夹中创建index.html。

    5. 右键打开Git Bash Hear。

    6. 输入git status(检测)。

    7. 输入git add .(缓存)。

    8. 输入git commit -m'（注释）'（提交版本库）。

    9. 输入git push（上传）。

    10. 上传邮箱。

    11. 上传用户名。

    12. 复制新手教程。

    13. 输入账号密码。

    14. 刷新页面，出现上传文件。

    15. 复制用户名+.github.io、打开新网页粘贴。

    # 2019年12月25日 第十七周 周三 实训第三天

    1. npm 【国外的下载】。

    2. cnpm【国内镜像】。

    3. 输入npm -v。

    4. 百度淘宝镜像，复制使用说明，粘贴至c盘根目录cmd。

    5. 等待下载，下载完成cmd中输入cnpm -v。

    6. 输入cnpm install（或者i） -g @vue/cli。

    7. 输入vue -V。

    8. 输入cnpm i -g vuepress。

    9. 百度vuepress。

    10. 建立工作文件夹，命名尽量用英文。

    11. 创建README.md文件。

    12. cmd命令打开文件夹。

    13. 输入vuepress dev 。

    14. 复制链接，打开浏览器。

    # 2019年12月26日 第十七周 周四 实训第三天

    1. 创建文件夹，命名为vuepress。

    2. 右键打开Git Bash Hear。

    3. 百度liujiaozhu666.git.io。

    4. 点击页面最下方indigo material。

    5. 复制模板分支命令、粘贴至Git。

    6. 运行init.sh。

    7. 后续运行文件在cmd中输入npm run dev。

    8. 复制链接百度打开。

    9. 修改方式:vuepress->docs->.vuepress->config.js。

    10. 可修改项:title、description、author、email、github。

    # 2019年12月26日 第十七周 周五 实训第五天

    1. 创建.md文件。

    2. 将文件拖入vuepress->docs->posh。

    3. 编辑md文件:

      ​ title: 标题 ​ date: 2019-12-27 23:32:03(时间及其格式) ​ tags: [类型]

    4. 下一行开始输入内容。

    5. 在文件夹cmd中输入npm run dev。

    6. 复制链接用浏览器打开。

    7. 在vuepress->node->deploy.sh中，第一行！前加#注释，最下方20行(git push -f...)删除#，修改地址。

    8. 在文件夹cmd中输入npm run build，打包完成。

    9. 双击文件夹中deploy.sh。

    10. 打开github官网查看是否打包上传成功。
